
一、项目结构（Project Structure）
Sales-Analysis-Project/
│── venv/                        # 虚拟环境
│ 
├── data/                        # 数据存储区
│   └── .....csv                 # 原始销售数据
│   └── .....xlsx                # 原始销售数据
│
├── notebooks/                   # Jupyter Notebook 脚本区
│   └── 01_data_loading.ipynb
│   └── 02_data_cleaning.ipynb
│   └── 03_outlier_handling.ipynb
│   └── 04_eda_visualization.ipynb
│   └── 05_summary_report.ipynb
│
├── outputs/                     # 输出结果区（图像和表格）
│   └── cleaned_data.csv
│   └── sales_by_region.png
│   └── summary_stats.csv
│   └── analysis_report.md       # Markdown 报告
│
├── scripts/                     # 可复用脚本函数库（可选）
│   └── data_utils.py            # 封装清洗、异常处理等函数
│   └── multi_file_processor.py   # 批量处理
│ 
│ 
│── sales_analysis_gui.py        #GUI
│── start_gui.bat                #一键启动脚本（首次建立虚拟环境并安装依赖项，之后自动激活虚拟环境检查依赖项并启动）
└── README.md                    # 项目说明文件（包含数据来源、步骤简要）

三、执行维度（Execution View）
1. 数据处理流程图
原始数据 → 缺失值填充 → 异常值处理 → 衍生字段（Sales） → 分组聚合 → 可视化 → 报告总结
2. 技术栈与工具
分类	工具/技术	用途
数据处理	pandas, numpy	清洗、计算
可视化	matplotlib, seaborn	柱状图、分组展示
环境	Jupyter Notebook, VSCode	脚本编写与运行
报告	Markdown、PDF、Word	项目输出格式
UI	plotly, streamlit	更高级的交互展示

五、交付成果汇总（Deliverables）
成果内容	                   文件位置	                                格式
清洗后的数据	outputs/cleaned_data.csv	CSV
柱状图图像	outputs/sales_by_region.png	PNG
描述统计表	outputs/summary_stats.csv	CSV
业务总结报告	report/analysis_report.md/pdf	Markdown 或 PDF
完整分析代码	notebooks/*.ipynb	Jupyter
